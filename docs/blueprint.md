# **App Name**: EduCompass

## Core Features:

- Universal Search: Enable users to search for institutions by keywords, location, and courses.
- Detailed Institution Profiles: Show exhaustive details for each institution, including academics, placements, and infrastructure.
- AI Institution Tool: Offer personalized institution suggestions using a Genkit-powered tool based on user-specified criteria (e.g., 'affordable engineering colleges in Maharashtra').
- Side-by-Side Comparison: Allow users to select up to three institutions and view their features in a comparison table.
- Favorites List: Let users save institutions to a dedicated 'favorites' page for easy access.
- Search History: Maintain a record of recently viewed institutions for easy revisiting.

## Style Guidelines:

- Primary color: Vivid blue (#29ABE2) to convey trust and guidance.
- Background color: Light blue (#E0F7FA) to create a calm and reliable atmosphere.
- Accent color: Soft green (#8BC34A) for calls to action and important information.
- Body and headline font: 'PT Sans', a humanist sans-serif for both headlines and body text due to its modern look and slight warmth.
- Utilize clear, modern icons to represent institution features and categories.
- Dashboard-style layout with cards for institution summaries, and a clean, responsive design.
- Subtle transitions and animations on search interactions to enhance user experience.
